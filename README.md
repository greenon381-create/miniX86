# miniX86

Experimental hobby operating system made from scratch in C.

## Features
- Bare-metal OS 
- Shell
- File editor
- Keyboard driver
- VGA text mode
- Works on x86 cpu PCs
- Easy to use

## Instruction to compile and run miniX86 OS (For Linux)

- cd to\your\folder_you are in
- make clean  # to clean the extra things in the folder (OS Scrap :) )
- make        # to compile and build the files
- make run    # to run the OS in Qemu if you have it
